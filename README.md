# Ecommerce-Store

This is an Ecommerce-Store-App.

You can:
- preview products
- change product category
- sort products
- filter products by 2 criterias - color and price
- see more items when you click Load More
- add to cart
- scroll to top

# 💻Technologies

 - ReactJS
 - Chakra UI
 - Cypress
